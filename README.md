# הכנסות והוצאות 2025

אפליקציית PWA למעקב הכנסות והוצאות, מגובה בגוגל שיטס.

## הגדרת גוגל שיטס

### שלב 1 – יצירת Apps Script

1. פתחי את [הגיליון שלך](https://docs.google.com/spreadsheets/d/1mFG-KWN6AgptS47rdfMgZRBWXyOwU6rp0M2NAqQLWNs/edit)
2. תפריט: **Extensions → Apps Script**
3. מחקי את כל הקוד הקיים
4. העתיקי את כל התוכן של `sheets-script.js` והדביקי
5. לחצי **Save** (Ctrl+S)

### שלב 2 – פריסה כ-Web App

1. לחצי **Deploy → New deployment**
2. בחרי סוג: **Web app**
3. הגדרות:
   - Execute as: **Me**
   - Who has access: **Anyone**
4. לחצי **Deploy** ואשרי הרשאות
5. **העתיקי את ה-URL** שמתקבל

### שלב 3 – חיבור לאפליקציה

פתחי את `index.html` ובשורה:
```js
const SHEETS_SCRIPT_URL = '';
```
הדביקי את ה-URL שהעתקת בין הגרשיים.

- ✅ הכנסות והוצאות בממשק מגע נוח

- ✅ סנכרון לגוגל שיטס
- ✅ PWA – ניתן להתקין על הטלפון
- ✅ עבודה אופליין (נתונים נשמרים מקומית)
